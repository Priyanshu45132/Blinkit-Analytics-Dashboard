# Blinkit-Analytics-Dashboard
Tools:- MS Excel &amp; Power BI

Built an interactive Power BI dashboard to analyze Blinkit’s sales performance, customer preferences, and outlet insights. The project focuses on key KPIs such as total sales ($1.2M), average sales, item count, and customer ratings.

The dashboard provides deep insights into:

Sales distribution by item type and fat content
Outlet performance based on size, location, and type
Trends in outlet establishment over time
Category-wise contribution to overall revenue

Designed with dynamic filters (location, outlet size, item type) to enable user-driven analysis and better decision-making. This project demonstrates strong skills in data visualization, data modeling, and business insight generation using Power BI.
